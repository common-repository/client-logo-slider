Showcase your client logos in an elegant, responsive carousel with the help of a shortcode.

== Description ==
Display a fully responsive client logo slider on any post or page using this shortcode:

`[wpcls3_logo_slider]`

**Fast and lightweight:** The scripts and stylesheets for the slider are only included on the pages where the shortcode is used.

**Customization options:** From the control panel (Plugins > Logo Slider Settings) you can modify each and every aspect of the slider including animation speed, number of items and autoplay.

**Completely responsive:** Works on all devices and screen sizes.

== Installation ==
1. For installation use the default plugin interface or drop the plugin files in your plugins folder.
2. Go to Plugins > Logo Slider Settings to modify the settings.
3. Insert the shortcode `[wpcls3_logo_slider]` in any post or page.
